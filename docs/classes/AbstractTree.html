appendChildNodeWithContent

  • appendChildNodeWithContent(parentNodeId: string, nodeContent: TGenericNodeContent<T>): string

  • Implementation of ITree.appendChildNodeWithContent

    • Defined in src/DirectedGraph/AbstractTree/AbstractTree.ts:60

    Appends a new child node with the provided content to a specified parent node.

    This method creates a new node in the tree, assigns it the given content, and makes it a child of the specified parent node. The node ID is automatically generated. The returned nodeId can be used in local scope to build up the expression. The tree manages content and may move it around. Although the nodeId may remain valid the content may get changed so nodeIds are considered temporal.

    Parameters

    • parentNodeId: string

      The ID of the parent node to which the new child node will be attached

    • nodeContent: TGenericNodeContent<T>

      The content to be stored in the new child node. Can be null, an object of type T, or another ITree

    Returns string

    The ID of the newly created child node
